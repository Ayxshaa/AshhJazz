# AshhJazz

A cinematic jazz editorial portfolio site with a noir aesthetic.

## Videos

The four scene videos were created using [Flow AI](https://flow.google) (Google's generative video tool), then embedded as the visual backbone of the site. Each scene captures a distinct mood — smoky, atmospheric, and shot through a 35mm film lens.

## What it is

A single-page Vue 3 app that presents the scenes as full-bleed editorial cards with scroll-driven reveal animations, a parallax layer, a film grain overlay, and a marquee ticker — all styled around a deep-red and amber noir palette inspired by 1960s jazz club photography.

## Stack

- **Vue 3** — component architecture
- **Vite** — build tooling
- **Custom composables** — `useParallax`, `useScrollReveal`

## Run locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
